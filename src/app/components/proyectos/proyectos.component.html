Aprosán
Información
Página web de Aprosán, asesoría online para personal sanitario. He utilizado el azul y el blanco como colores corporativos, algo muy común en páginas web del sector sanitario. Para darle un toque más fresco y moderno, he jugado con la opacidad y los gradientes. También he incluido animaciones, todo ello con el fin de mejorar la experiencia de usuario.
Además de diseñar la página web, diseñé también el logotipo. He utilizado un fonendoscopio para sustituir la letra inicial del nombre de la compañía. Por otro lado, su extremo viene a emular una tilde. Me ha parecido apropiado, dado que este dispositivo se utiliza para escuchar los latidos del corazón. Con ello, quería expresar protección. Para mantener la coherencia con el color corporativo y el logotipo, he utilizado imágenes en las que el azul y el blanco estuvieran presentes o en las que apareciera un fonendoscopio.

Angular Bootstrap HTML SASS TypeScript JavaScript jQuery Git
SententiApp
Información
Página web de SententiApp, proyecto de investigación del Instituto Universitario de Análisis y Aplicaciones Textuales (IATEXT) de la Universidad de Las Palmas de Gran Canaria (ULPGC). Su cometido es ofrecer un panel de administración que permita a los investigadores y lingüistas incorporar datos en el sistema que posteriormente sustentarán una app móvil a través de una API. El color corporativo es el dorado. Para no congestionar la interfaz, hemos hecho un gran uso del color blanco. La app móvil la puedes encontrar en este catálogo de proyectos.
Además de diseñar la aplicación web, diseñé también el logotipo. Encima del texto vemos una pluma escribiendo en un teléfono móvil, dado que se trata de una app móvil en la que se encuentran sentencias escritas por autores clásicos, que comúnmente escribían con pluma. El texto utiliza una tipografía gruesa con sefira. El teléfono móvil y el gradiente hacen de elementos modernos, mientras que la pluma y la serifa hacen de elementos con un look más antiguo. Esta combinación nos ofrece una mayor holgura a la hora de diseñar la interfaz web.

.NET C# Bootstrap HTML SASS JavaScript jQuery Microsoft Azure Inkscape

Portfolio online de Soledad Poveda Montesdeoca, asistente legal. Es un regalo de cumpleaños que le hice a una de mis hermanas para ayudarla a conseguir trabajo. Con ello pretendo que a la hora de presentarse a una oferta de trabajo, pueda incluir un link a esta página web y diferenciarse, de este modo, del resto de posibles candidatos, al contar con una mayor presencia web. De esta manera se ofrece la información necesaria en un formato más moderno, interactivo y amigable que el formato convencional ofrecido por una hoja de papel o un fichero PDF.

El blanco está muy presente para ofrecer cierto alivio visual. En varias secciones aparecen imágenes sutilmente visibles relacionadas con el área del Derecho.
Además de diseñar la página web, diseñé también el logotipo. El símbolo que aparece encima del texto es una S mayúscula (la inicial del nombre de mi hermana) que se combina con una columna griega enrollándose por sus extremos a modo de pergamino. Mediante el dibujo intento darle un aspecto clásico. No obstante, utilizo una tipografía moderna con serifa para evitar que el conjunto se viera anticuado. El uso de elementos griegos en el símbolo se debe a que el Derecho Griego fue el precursor intelectual y la fuente del Derecho Romano.

JC Abogada
Información
Página web de JC Abogada. En ella podremos encontrar información acerca de Jessica Carmada, así como los servicios ofrecidos en su despacho. Si bien es cierto que la estétita de la web no evoca la misma seriedad que encuentras en otras páginas web de despachos de abogados, al tratarse de un blog y de un despacho unipersonal, quise darle un aspecto personal, cercano, cálido y diferente (a veces es bueno salirse de lo convencional). El color corporativo es salmón. Las fotos muestran elementos de oficina en los que el color corporativo se mantiene presente.
Al tratarse de un blog, he utilizado WordPress. Existe una separación de usuarios para evitar que el cliente pueda realizar tareas de administración que vulneren la seguridad o alteren el aspecto de la página web, dado que el cliente desea encargarse de la publicación de noticias. Como tema he utilizado Enfold, cuyo page builder es Avia, dado que he trabajado previamente con él y su page builder es de los más manejables que he probado. No obstante, este no ofrece demasiadas herramientas de maquetación, pero permite asignar clases CSS a los elementos que componen la estructura del sitio, de modo que he escrito bastante código SASS para dejar la web a mi gusto. Asimismo, los modales que aparecen al hacer clic en cualquiera de las áreas de actuación fueron implementados con JavaScript. Por seguridad, he trabajado con un tema hijo para evitar la pérdida de información tras actualizaciones y facilitar el mantenimiento.
Además de diseñar la página web, diseñé también los dos logotipos: uno más compacto y otro más grande y decorativo. Este último utiliza una tipografía principal fina que intenta emular una caligrafía. He alargado sus extremos y obviado las mayúsculas para darle un aire desenfadado y moderno. La tipografía secundaria carece de serifa y es más fina para evitar así sobrecargar el logotipo. También me encargué de la imagen corporativa en Facebook, así como del diseño de las tarjetas de contacto.

Nido de Pío Pío
Información
Página web de El Nido de Pío Pío, guardería. No se cuenta con un solo color corporativo. Se hace un uso excesivo de muchos colores y gradientes para trasmitir desorden, diversión rebeldía, evocando con ello la infancia. La tipografía utilizada emula una caligrafía infantil, pero sin sacrificar la legibilidad. El sitio cuenta con una galería de imágenes de tipo Masonry y de un carrusel para ilustrar los servicios ofrecidos. He creado diferentes bordes para separar las secciones entre sí emulando olas o dándole un efecto brush. Asimismo, las imágenes seleccionadas son muy coloridas y mezclan juguetes con naturaleza.
